# Insta Desktop Clone

A Instagram (Desktop version) clone I made to improve my Web Dev skills.

# Description

Works as an image sharing website similar to to Instagram (doesn't have any of the facny compression stuff).

## Dependencies
* Express
* Express Session
* Mongoose
* BodyParser
* Mehod Override
* Passport
* Passport Local
* Passport Local Mongoose
* EJS

* JavaScript
